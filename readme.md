# 支持1024*768分辨率VGN输出的CPU
## RISC-V
    支持指令：
    add
    sub
    and
    or
    slt
    sltu
    addi
    andi
    ori
    xori
    slti
    slli
    srli
    srai
    lw
    sw
    ecall
    beq
    bne
    jal
    jalr
    xor
    auipc
    lhu
    bgeu
    blt
    lui
    mul
    div

## VGN
    将内存0x008开始的32*24个字节映射到屏幕的1024*768个像素点
